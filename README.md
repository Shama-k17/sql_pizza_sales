🍕 Pizza Sales Analysis (SQL Project)
📌 Overview

This project analyzes pizza sales data using SQL to uncover key business insights such as top-selling pizzas, revenue trends, and category performance.

🧰 Tools Used
MySQL Workbench
SQL
📊 Dataset

The dataset includes:

Orders
Order Details
Pizzas
Pizza Types
🔍 Key Insights
Top pizzas contribute significantly to overall revenue
Certain categories (like Classic & Chicken) dominate sales
Revenue shows consistent growth over time
🚀 SQL Concepts Used
Joins
Aggregations (SUM, COUNT)
Subqueries
Window Functions (RANK, OVER)

📬 Conclusion

This project demonstrates practical SQL skills for data analysis and extracting meaningful business insights.
